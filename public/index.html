Las frases celebres del Prof. Charlie

Ejercicio Formas y Metodo POST

Duración: 2:30
  1. Crear una app de React
  2. Deberan obtener los valores validos para las clases desde el siguiente endpoint /api/quotes/options
  3. Crear un componente llamado Quote el cual deberán mostrar una cita con la clase en la que el profesor la dijo
  4. En el componente App deberan traer todas las citas desde el sig endpoint /quotes - GET
  5. Deberan crear un Map y renderizarlas
  6. Deberar crear un componente llamado CreateQuote el cual incluya un formulario con los siguientes campos: quote && class
  7. El campo quote debera recibir la cita que quieran incluir
  8. El campo class debe ser de tipo select y generar las options en base al paso 2
  9. Al dar submit deberan usar axios con el metodo POST al siguiente endpoint /quotes
  10. Actualizar el estado en App con la nueva cita
  11. El componente Quote, puede tener un botón, icono, etc que al dar click borre la cita del servidor usando el metodo DELETE
Retos opcionales
  1. Reutilizar el componente CreateQuote para el metodo PUT (actualizar la cita)
